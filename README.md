# Braukaiser Archive

This repository is an archived copy of the **braukaiser.com** wiki, a valuable resource for the homebrewing community that focused on German brewing techniques and science.

## Disclaimer

**I do not own this content.** All articles, images, and documents in this repository were originally created by **Kai Troester** and published on [braukaiser.com](http://braukaiser.com). This archive was created solely to preserve this educational resource for the brewing community as the original site became unavailable.

## About the Original Site

Braukaiser.com was a comprehensive wiki covering:

- German brewing science and techniques
- Decoction mashing procedures
- Water chemistry and pH control
- Malt and enzyme science
- Traditional German beer recipes
- Yeast culturing and fermentation
- And much more
